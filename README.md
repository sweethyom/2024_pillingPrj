# プロジェクト概要
2024.02.19-2024.03.20（30日間）

# 企画背景
新型コロナウイルスのパンデミック以降、健康への関心が世界的に高まり、特に20-30代の若い世代における個別化された健康ソリューションの需要が急増しています。
このような背景から、私たちは20-30代をメインターゲットとしたサプリメント推奨および購入サービスを企画することになりました。

# 目標
本サービスは、アンケート調査によるカスタマイズされたサプリメント推奨プロジェクトで、若い世代の健康ニーズを把握し、個別化された栄養ソリューションを提供することを目標としています。

# サービス名
Pi1ling
：あなただけのための唯一無二のサプリメントショッピングモール

# 主要機能
## アンケート調査
  - ユーザーの生活パターンおよび健康状態を把握
  - 選択完了時、提示された属性に該当するオプションを持つサプリメントをフィルタリングして推奨
## サプリメント商品検索
  - サプリメント商品の検索が可能

## サプリメントレビュー機能
  - サプリメントレビューの作成および履歴

## サプリメント商品
  - サプリメントの詳細ページおよびリスト

## ショッピングカート

## マイページ

  - レビュー履歴
  - お気に入り履歴

# 프로젝트 개요
2024.02.19-2024.03.20(30일)

# 기획 배경
코로나19 팬데믹 이후 건강에 대한 관심이 전 세계적으로 증가하며, 특히 20-30대 젊은 층의 개인화된 건강 솔루션 수요가 급증하고 있습니다.

이러한 배경에서 저희는 20-30대를 주 타겟으로 하는 영양제 추천 및 쇼핑 서비스를 기획하게 되었습니다. 


# 목표
본 서비스는 설문조사를 통한 맞춤형 영양제 추천 프로젝트로, 젊은 세대의 건강 니즈를 파악해 개인화된 영양 솔루션을 제공하는 것을 목표로 합니다.

# 서비스명
## Pi1ling
### : 당신만을 위한 단 하나의 영양제 쇼핑몰
